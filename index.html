About The Project

This crypto portfolio web app is a pet project that I created to showcase my skills and understanding of various technologies. It is one of several projects that I have worked on, and you can view my full portfolio here.

I built this app using the following technologies:

  • HTML
  • SCSS
  • Vanilla JS

So far, I have completed the Home, Market, and Learn sections of the app, providing users with the ability to track multiple cryptocurrencies, view market data, and learn about investing in crypto. I am continuously working to improve the app and plan to add new functionality in the future.
