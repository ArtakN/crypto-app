About The Project

CryptoView is a web application that I developed from scratch to showcase my skills and understanding of various technologies. No ready-made solutions were used in its creation, demonstrating my ability to build complex systems independently. It is one of several projects that I have worked on. You can view my full portfolio here.


Technologies Used


I built this app using the following technologies:
  • HTML
  • SCSS
  • Vanilla JS

Current Features


The application allows to engage with real-time data of the top 50 cryptocurrencies, sourced directly from CoinGecko. The market list can be filtered by coin name, coin price, 24h price change, and market cap for easy navigation. The list is also paginated for user convenience. Additionally, the application includes a dark/light mode toggle, enhancing accessibility by allowing users to switch between themes based on their preference.
